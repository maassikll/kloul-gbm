Langage c++ :

Le C++ est un langage de programmation aux nombreux paradigmes : orienté objet, procédural ou générique. Le C++, au même titre que le C, est un langage de bas niveau. C’est-à-dire qu’il est plus proche du fonctionnement de la machine. Le principal cas d’utilisation du C++ est aujourd’hui le suivant : Besoin de développer une application où la performance est primordiale Voici l’explication sur l’origine de son nom : C vient du langage de programmation C datant de 1970. Le suffixe “++” correspond à l’opérateur permettant d’incrémenter une valeur en C. Cela signifie donc que C++ se positionne comme une version améliorée et plus performante que le langage C. Le langage C++ est régulé par la norme ISO afin de le standardiser. La dernière version C++ 20 est sorti en 2020.Moi je suis fan dans son utlisation en Arduino en particulier

Example

Tout comme les fichiers C, les fichiers CPP contiennent du code source écrit en langage de programmation orienté objet C++. Étant donné que les informations contenues dans les fichiers CPP sont stockées en texte brut, ces derniers peuvent être ouverts par n’importe quel éditeur de texte. Toutefois, pour tirer le meilleur parti du code, ils doivent être ouverts dans un programme qui prend en charge la coloration syntaxique du code C++ contenu. Les fichiers CPP sont très connus du fait de la popularité du C++.le mieulleur IDE a utilser est celui fournit par le site officielle Arduino.cc https://www.arduino.cc/

Ici dans se projet on a utiliser un carte Arduino UNO qui est bon marcher.Elle est munis de deux types d'entrées Dégitales et Analogique

Le matériels requis :

  • carte Arduino UNO
  • Leds
  • Capteur de chaleur TMP36
  • Ecran LCD 16*2
  • Plaque d'essaie
  • Résistances,potentiométre
  • Un button poussoir
